Project overview

As developer working for a 3D studio, I was tasked with creating an app to automate the process of managing files and folders. Using the .NET framework and C# programming language, I developed a file manager app that allows users to easily upload files and folders and perform various actions on them.

The app also includes advanced features such as batch processing, which allows users to perform the same action on multiple files or folders at once, and search functionality, which makes it easy to locate specific files or folders within a large directory.

Overall, this file manager app has been a valuable asset for my team, streamlining their workflow and allowing them to automate many time-consuming tasks. Its user-friendly interface and powerful functionality make it an excellent tool for anyone who needs to manage large quantities of files and folders on a regular basis.
